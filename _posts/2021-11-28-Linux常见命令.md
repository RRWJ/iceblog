### top命令：
```
输入格式：$ top
```
**Note:** 实时监控系统负载、CPU运行情况。       
#### q：退出基本视图
#### b：打开/关闭加亮效果
#### x；打开/关闭排序列的加亮效果
#### f：进入另一个视图

### 软连接：
```
$ ln -s 源地址 目的地址
```
**Note:** 相当于将*源地址目录下的所有文件*在*目的地址*中创建快捷键，类似指针操作。

### 移动文件：
```
$ mv 源地址 目的地址
```

### 返回根目录：
```
$ cd /
```

### 配置软连接

比如微信或者其他软件，下载的文件深度很深，不能被搜索到。这个时候怎么办喃，建立软连接配置路径。
 比如，文件在LIbrary下面，一般情况是搜索不到的，在Download目录下创建一个目录，然后创建一个软连接到需要路径下面。这样搜索的就可以立马触达。
